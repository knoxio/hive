## User Story

As a user, I want Hive to work well on mobile and tablet devices so that I can monitor rooms and agents on the go.

## Complexity
M (Medium)

## Priority
P2

## Dependencies
- App shell

## Acceptance Criteria
- [ ] Sidebar collapses below a defined breakpoint (e.g., 768px)
- [ ] Hamburger menu to toggle sidebar on mobile
- [ ] Touch-friendly tap targets (minimum 44px)
- [ ] Panels stack vertically on mobile (no horizontal split)
- [ ] Message input remains accessible and usable on mobile
- [ ] No horizontal scrolling on any viewport width

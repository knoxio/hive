## User Story

As a user, I want to filter rooms in the sidebar by name so that I can quickly find a specific room when the list is long.

## Complexity
S (Small)

## Priority
P2

## Dependencies
- Room list

## Acceptance Criteria
- [ ] Search/filter input at the top of the room list in the sidebar
- [ ] Rooms filtered by name as user types (case-insensitive)
- [ ] Clear filter button (X icon) to reset
- [ ] Empty state message when no rooms match the filter
- [ ] Filter does not affect unread badges or room ordering logic

## User Story

As a user, I want to send and receive direct messages through the web interface so that I can have private conversations with specific users.

## Complexity
M (Medium)

## Priority
P2

## Dependencies
- Chat system
- User list

## Acceptance Criteria
- [ ] DM button on user profile/card or user list entry
- [ ] Dedicated DM conversation view (separate from room chat)
- [ ] DM notifications (browser notification + badge)
- [ ] DM list in sidebar (separate section from rooms)
- [ ] DM messages use the existing wire format DirectMessage type
- [ ] DM history persisted and loadable
- [ ] Online/offline indicator in DM conversation

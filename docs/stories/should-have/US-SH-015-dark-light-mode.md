## User Story

As a user, I want to toggle between dark and light themes so that I can use Hive comfortably in different lighting conditions.

## Complexity
S (Small)

## Priority
P3

## Dependencies
- Tailwind CSS setup

## Acceptance Criteria
- [ ] Toggle button/switch in header or settings
- [ ] Theme preference persisted to localStorage
- [ ] Respects system preference (prefers-color-scheme) on first visit
- [ ] All UI components render correctly in both themes
- [ ] Smooth transition between themes (no flash of unstyled content)

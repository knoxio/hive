## User Story

As a user, I want keyboard shortcuts for common actions so that I can navigate and operate Hive efficiently without a mouse.

## Complexity
M (Medium)

## Priority
P3

## Dependencies
- App shell
- Routing

## Acceptance Criteria
- [ ] Ctrl+K (or Cmd+K) opens command palette
- [ ] Ctrl+1 through Ctrl+4 switches between main tabs/sections
- [ ] Ctrl+Enter sends message in chat input
- [ ] Escape closes modals and overlays
- [ ] ? key shows keyboard shortcut help overlay
- [ ] Shortcuts do not conflict with browser defaults
- [ ] Shortcuts are disabled when typing in input fields (except Ctrl+Enter)

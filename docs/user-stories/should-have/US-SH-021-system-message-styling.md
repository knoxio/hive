## User Story

As a user, I want system messages (join, leave, events) to be visually distinct from chat messages so that I can quickly differentiate coordination events from conversation.

## Complexity
S (Small)

## Priority
P3

## Dependencies
- Chat timeline

## Acceptance Criteria
- [ ] Join/leave messages styled with muted/subtle appearance
- [ ] Event messages (task claimed, task finished, etc.) styled with distinct icons
- [ ] System messages centered or indented differently from user messages
- [ ] Timestamp-only display for repetitive system events
- [ ] Icon indicators for different event types (join, leave, task, error)
- [ ] System messages are collapsible when many occur in sequence

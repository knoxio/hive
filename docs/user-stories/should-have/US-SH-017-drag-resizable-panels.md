## User Story

As a user, I want to resize panels by dragging dividers so that I can customize the layout to my preferences.

## Complexity
M (Medium)

## Priority
P3

## Dependencies
- App shell

## Acceptance Criteria
- [ ] Drag handles (dividers) between adjacent panels
- [ ] Panels resize smoothly while dragging
- [ ] Panel sizes persist to localStorage across sessions
- [ ] Minimum panel widths enforced (panels cannot be collapsed to zero)
- [ ] Double-click on divider resets to default sizes
- [ ] Cursor changes to resize indicator on hover over divider
